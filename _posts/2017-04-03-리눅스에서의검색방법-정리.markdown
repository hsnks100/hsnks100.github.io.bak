# 파일 검색

```
find . -name "apple"
```

. 현재 위치로 부터 파일명이나 디렉토리명에 apple 이라는 글자가 포함된 것을 찾아줌.

# 파일내 검색 

```
grep -r "def" ./
```

현재 위치로 부터 모든 파일내용에 대해 def 라는 글자가 들어가있는것을 찾아줌.

여기서 "" 안에 있는 찾는 문자열은 정규식을 사용할 수가 있음.
